

# 컨설팅 산출물 지식검색 위키 — 프론트엔드 UI

## 프로젝트 개요
Google Drive 동기화 폴더의 PPT/PDF/Sheets/CSV/ipynb 등 파일 내부 텍스트를 "파일명 + 위치(슬라이드/페이지/시트/행) + 스니펫"으로 즉시 검색할 수 있는 **사내 지식 위키 웹 UI**를 구축합니다.

> ⚠️ **Lovable 범위**: 프론트엔드 UI만 구축합니다. 백엔드(파이썬 인덱서, Meilisearch, FastAPI)는 별도로 구축해야 합니다. UI는 Mock 데이터로 즉시 데모 가능하며, API baseUrl만 바꾸면 실제 백엔드와 연결됩니다.

---

## 구현할 페이지 및 기능

### 1. 메인 검색 페이지 (Home)
- **상단 검색바**: 키워드 입력 → 엔터로 검색
- **검색 결과 리스트**: 파일명, 경로, 위치(슬라이드/페이지/시트/행), 스니펫 하이라이트
- **좌측 필터 패널**: 프로젝트 경로, 파일 타입(pptx/pdf/xlsx/csv/ipynb), 정렬(관련도/최근수정)
- **결과 클릭 → 상세 패널**: 해당 chunk 전체 텍스트 + 메타데이터 표시
- **경로 복사 버튼**: 클립보드에 파일 경로 복사

### 2. Browse (폴더 탐색) 페이지
- **폴더 트리 뷰**: 프로젝트/사업 폴더 구조를 트리로 탐색
- **해당 폴더 내 파일 리스트**
- **"이 폴더 내 검색" 토글**: 선택한 폴더 범위로 검색 제한

### 3. 인덱싱 현황 대시보드
- 총 파일 수, 총 chunk 수, 최근 업데이트 시각
- 파일 타입별 통계 (pptx/pdf/xlsx 등 비율)
- 인덱싱 실패 파일 목록

### 4. 문서 상세 페이지
- 해당 파일의 모든 chunk 리스트 (슬라이드/페이지/시트별)
- chunk 내 텍스트 전문 + 위치 정보
- 파일 열기/경로 복사

---

## 디자인 방향
- **업무용 깔끔한 UI**: 사내 도구 느낌, 검색과 결과 가독성 최우선
- 좌측 사이드바 네비게이션 (검색 / Browse / 대시보드)
- 검색 결과는 카드/테이블 혼합 레이아웃
- 하이라이트 강조로 키워드 빠르게 스캔 가능

## 데이터 구조
- Mock 데이터로 컨설팅 산출물 샘플 (PPT 슬라이드, PDF 페이지, Excel 시트 등) 포함
- API 서비스 레이어를 분리하여, `baseUrl` 환경변수만 바꾸면 실제 FastAPI 백엔드로 즉시 전환 가능
- API 엔드포인트: `GET /search`, `GET /doc/{chunk_id}`, `GET /stats`, `GET /browse`
